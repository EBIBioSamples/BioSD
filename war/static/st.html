SampleTab format

Note: Specification of this file format is at an early stage. All details are subject to flux, depending on how useful they are in the face of real data.

The SampleTab format is an easy-to-use spreasheet-like format for describing collections of biological samples. Its main purpose is for submission to the BioSamples Database at EBI.

The documentation about the SampleTab format is divided into three sections, each aimed at a different type of user.

For Biologists

This documentation is for the wet-lab biologist who wants the simplest way to submit a small number of samples. It describes the basic structure of the file with examples and a fill-in-the blanks template.

For submitting Bioinfomaticians

In addition to the documentation for Biologists, this documents how to describe more unusual collections of samples. For example, groups of samples, confidential samples, samples derived from other samples, linking samples to other databases, etc.
